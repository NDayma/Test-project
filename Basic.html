  • In the header there are various meta attributes
    1. Charset: Assign "UTF-8" as value to it
    2. name and content: in name assign viewport and content "width = device-width". This aspect makes page responsive by providing width value as device width which implies that based on the device on which page is opened the display of the content will be adjusted accordingly.

  • HTML comments are written in format <!-- and --> Anything between these two thing is considered as comment.
  • Meta tags
    1. http-equiv: This meta type has attribute http-equiv and content. Content is necessary attribute. The value assigned to http-equiv and content determines how the page is going to be rendered. In case of http-equiv being set to refresh and content 60 implies page gets auto refreshed in 60 seconds. The most common used is < meta http-equiv="content-security-policy" content="default-src https:" />. content-security-policy determines the permitted server origin and script end points.
    2. Name attribute: The other type of attribute is "name" attribute which has values viewport,description and theme-color. It also needs to have content as attribute along with the name attribute.
      • The name attribute in meta tag can carry description as value. The syntax for the same is < meta name="description" content="register for learning" />. The description value of attribute determines your rank in SEO. It should be short and precise description of the page.
      • Robots is another name attribute which states bots to not index the site and follow any links
      • The theme-color attribute determines the browser color at the place where url is written.
    3. open graph protocol: The another attribute used in header is og attribute. This attribute determines what content to be displayed when the website is referred on other social networking sites. more about it can be read on https://ogp.me/.
      <meta property="og:title" content="Machine Learning Workshop" />
      <meta property="og:description" content="School for Machines Who Can't Learn Good and Want to Do Other Stuff Good Too" />
      <meta property="og:image" content="http://www.machinelearningworkshop.com/image/all.png" />
      <meta property="og:image:alt" content="Black and white line drawing of refrigerator, french door refrigerator, range, washer, fan, microwave, vaccuum, space heater and air conditioner" />
    4. <meta name="apple-mobile-web-app-capable" content="yes" />
      <meta name="mobile-web-app-capable" content="yes" /> .In this meta tag when we pass the content type to yes will result the application will run in full screen mode, without the address bar at the top and the navigation bar at the bottom. In case of no, the address bar and navigation bar will shown.
